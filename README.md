# 🧮 计算器 Calculator

一个使用 HTML + CSS + JavaScript 制作的简洁计算器。

## 📸 预览

打开 `index.html` 即可使用！

## 🚀 如何使用

1. 下载项目
2. 双击 `index.html` 打开
3. 开始计算！

## 🎯 功能按钮

| 按钮 | 功能 |
|------|------|
| C | 清空所有 |
| ⌫ | 删除最后一位 |
| % | 百分比 |
| ÷ × - + | 四则运算 |
| = | 计算结果 |
| 0-9 | 数字输入 |
| . | 小数点 |

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| 0-9 | 输入数字 |
| + - * / | 运算符 |
| Enter 或 = | 计算 |
| Escape | 清空 |
| Backspace | 删除 |

## ✨ 功能特点

- ✅ 简洁美观的界面
- ✅ 支持加减乘除
- ✅ 支持百分比计算
- ✅ 支持键盘输入
- ✅ 响应式设计
- ✅ 无需安装，浏览器直接运行

## 🛠️ 技术栈

- HTML5
- CSS3
- JavaScript (原生)
